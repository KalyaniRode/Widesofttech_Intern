Web Technologies

HTML

HTML (HyperText Markup Language) is used to define the structure of web pages using tags like <div>, <p>, <a>, etc.

CSS

CSS (Cascading Style Sheets) is used to control the appearance and layout of HTML elements such as colors, fonts, spacing, and positioning.

Bootstrap

Bootstrap is a CSS framework that provides pre-built components and utility classes to create responsive and mobile-friendly websites quickly.
