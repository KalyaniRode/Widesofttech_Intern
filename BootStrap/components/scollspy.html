HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the Web. It tells the browser what elements to display on a page — such as text, images, links, headings, tables, forms, and more.

CSS

CSS (Cascading Style Sheets) is the language used to style and design the layout of web pages. It controls how HTML elements look — such as their color, size, position, font, background, spacing, and more..

Bootstrap

Bootstrap is a free, open-source CSS framework that helps you build responsive, mobile-first websites quickly. It provides ready-to-use components like buttons, forms, navbars, modals, grids, and more — without writing custom CSS..
