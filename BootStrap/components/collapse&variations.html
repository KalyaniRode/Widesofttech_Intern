Bootstrap Collapse Variations

Basic Collapse

Bootstrap Collapse is used to show and hide content dynamically with smooth animation using JavaScript and CSS transitions.

Horizontal Collapse

Horizontal collapse expands or contracts width-wise instead of height-wise. It's useful for side panels or menus.

Multiple Collapse Sections

Use data-bs-toggle="collapse" with a data-bs-target attribute pointing to the ID of the element to collapse.

Supports animation, toggle from multiple buttons, and can be grouped with accordions or horizontal layout.
